<h1>BaseKernel</h1>

<p> IS A 32 BIT OPERATING SYSTEM FROM SCRATCH MADE BY Huguini79 TO SHOW PEOPLE HOW TO CREATE AN OPERATING SYSTEM FROM SCRATCH WITH A LOT OF DETAILS, LIKE A LOT OF COMMENTS IN C OR IN X86 ASSEMBLY</p>

<img width="747" height="243" alt="Screenshot 2025-12-13 21 32 37" src="https://github.com/user-attachments/assets/40764e62-b25c-4cd7-a032-6a76ad449543" />

<p>
<h2>=======DIRECTORIES========</h2>
<p>/build --> All the object files are here</p>
<p>/src ---> All the source code files are here</p>
<p>/iso | iso/boot | iso/boot/grub --> The GRUB configuration file and the elf kernel file are here</p>
<p>/bin --> The final ISO image of the operating system is here</p>

<h2>=====FILES=====</h2>
<p>build.sh --> The build file</p>
<p>src/linker/linker.ld --> The linker file</p>
<p>iso/boot/grub/grub.cfg --> The GRUB configuration file</p>
<p>src/kernel/kernel.asm --> The entry point from GRUB and the C kernel file</p>
<p>src/kernel/kernel.c --> The C kernel file</p>
<p>src/kernel/kernel.h --> The C kernel header file</p>
<p>src/vga.h --> The VGA header file</p>
<p>src/vga/vga.c --> The VGA implementation in C from scratch</p>
<p>src/io.h --> The I/O header file</p>
<p>src/io/io.asm --> The I/O implementation in X86 Assembly from scratch</p>
<p>src/keyboard.h - The Keyboard header file</p>
<p>src/io/keyboard/keyboard.c - The Keyboard implementation in C from scratch</p>
</p>
